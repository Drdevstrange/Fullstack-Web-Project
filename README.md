This is a full-stack social media web application built using Django, designed with scalable architecture principles and cloud-ready deployment in mind.
The platform enables users to create profiles, share posts, interact with content, and engage socially through real-time-like interactions powered by AJAX.
This project demonstrates backend development, database design, RESTful architecture principles, frontend integration, and cloud-based deployment concepts.

Please use the command : pip install -r requirements.txt
to install all the required packages
